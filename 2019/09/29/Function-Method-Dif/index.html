[ETC] 함수와 메소드의 차이

2019-09-29
ETC, 메소드, 함수, 함수와 메소드 차이

개발을 하다보면 함수와 메소드라는 말을 자주 접하고 듣는다. 어찌보면 개발자에게 있어서 없어서는 안되는 것이기도 하다. 그런데 어떤 상황에서는 함수라고 하고 어떤 상황에서는 메소드라고 한다. 둘의 차이가 뭔지 궁금해서 해당 글을 포스팅해본다.

함수(Function)

  • 특정 작업을 수행하는 코드 조각이다.
  • 전역이던 지역이던 독립된 기능을 수행하는 단위다.
  • 함수가 메소드를 포함하고 있다.
  • 함수가 더 포괄적인 개념이다.

메소드(Method)

  • 클래스, 구조체, 열거형에 포함되어 있는 함수를 메소드라고 부른다.
  • 다른말로 클래스 함수라고도 한다.
